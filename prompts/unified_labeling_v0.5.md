# 통합 라벨링 지침 (Unified Labeling Strategy v0.5)

본 지침은 한국어 독성 텍스트 데이터셋 구축을 위한 **최종 가이드라인**이다. AI 에이전트는 본 지침을 헌법처럼 준수하며, 특히 **데이터 무결성 보존**과 **제3자적 관점(훈계/평론) 배제**를 최우선 가치로 삼는다.

---

## 🚦 [최우선] 데이터 무결성 및 금기 사항 (Strict Rules)

1. **origin_tags 절대 무결성**: `origin_tags` 컬럼은 원본(`unsmile_train.tsv`)의 라벨을 **토씨 하나 틀리지 않고 그대로 복제**한다. (예: `여성/가족, 남성`을 `여성/남성`으로 줄이거나 `, clean`을 함부로 붙이지 말 것)
2. **제3자 평론 및 훈계 금지**: `target` 컬럼에 "안타깝습니다", "예의가 없군요", "반성하십시오", "응원합니다" 등 AI의 평가나 감정을 단 한 마디도 섞지 않는다.
3. **직접 번역 원칙**: 에이전트는 원문의 화자가 되어 **정중한 서울 표준어**로 다시 말하는 것처럼 번역한다. (예: "비판하시네요" -> (X), "비판합니다" -> (O))

---

## 1. 데이터 스키마 (7-Column Schema)

| 컬럼명 | 설명 | 규칙 |
| :--- | :--- | :--- |
| **source** | 입력 원문 | **수정 절대 금지** |
| **target** | 최종 정제 출력 | Action 코드에 따른 규칙 적용 (하단 참조) |
| **action** | 판정 코드 | **0: NORMAL, 1: BLOCK, 2: PURIFY** |
| **severity** | 유해 수위 | 0(안전) ~ 5(극히 유해) |
| **category** | 표준 유형 | GENDER, LGBTQ, RACISM, REGIONAL, COMMUNITY, AGE, RELIGION, SLUR, NORMAL |
| **reason** | 판단 및 순화 근거 | `[핵심단어/맥락] -> [판단근거/순화방향]` 형식 |
| **origin_tags** | 원본 태그 정보 | **원본 데이터셋의 값을 100% 보존** |

---

## 2. 액션 판독 및 출력 규칙 (Action Logic)

### 2.1 Action 0: NORMAL (유지)
- **적용**: 무해한 일상 대화, 단순 사실, **정상적인 방언**.
- **출력**: `source`와 100% 동일하게 유지.

### 2.2 Action 1: BLOCK (차단)
- **적용**: 범죄 선동, 물리적 폭력 위협, 살인/자살 종용, 반인륜적 성범죄 묘사, 극단적 혐오.
- **출력**: **(차단된 문장입니다)**

### 2.3 Action 2: PURIFY (순화)
- **적용**: 욕설, 비하어, 특정 집단 조롱, 커뮤니티 희화화 말투.
- **출력**: 원문의 의도를 유지하되 **정중하고 담백한 서울 표준어**로 직접 번역.

---

## 3. 사투리 vs 커뮤니티 어투 판별 (-노 규칙)

- **NORMAL (Action 0)**: **의문사(뭐, 와, 어데, 누구 등)**가 포함된 자연스러운 방언 의문문.
- **COMMUNITY (Action 2)**: 의문사 없는 평서문/판정 의문문 끝의 '-노', 인터넷 밈 종결어미(-이기, -노데스웅), 특정 커뮤니티 전용 비하 은어.
- **판단 보수주의**: 사투리인지 조롱인지 모호할 경우, **COMMUNITY(Action 2)**를 선택하여 표준어로 순화한다.

---

## 4. 세부 순화 원칙 (Standardization)

1. **1:1 성분 대응**: 문장의 주어, 목적어, 술어 구조를 가급적 유지한다.
2. **담백한 문어체**: 가급적 "~합니다", "~입니다", "~입니까?"와 같은 격식 있는 서울 기준 표준어를 사용한다.
3. **의소 보존**: 원문의 감정(분노, 냉소, 의문)은 유지하되 표현의 '그릇'만 바꾼다.
   - *예: "존나 싫다" -> "매우 싫습니다" (O) / "좋지 않게 생각하시는군요" (X)*
4. **메타 데이터 배제**: 원문에 없는 배경 설명이나 부연 설명을 추가하지 않는다.

---

## 5. 유형별 카테고리 (Category)
- **GENDER**: 성별 비하.
- **LGBTQ**: 성소수자 비하.
- **RACISM**: 인종/국적/외국인 비하.
- **REGIONAL**: 특정 지역민/장소에 대한 지리적 차별 및 혐오.
- **COMMUNITY**: 특정 인터넷 커뮤니티 발 말투, 밈, 변칙적 어법에 의한 조롱.
- **AGE**: 연령 비하 (틀딱, 잼민이 등).
- **RELIGION**: 종교 및 신도 비하.
- **SLUR**: 특정 집단이 아닌 보편적인 악플, 저급한 표현.
- **NORMAL**: 유해 요소 없음.
